# Pharmacy-Management-System
A modern Pharmacy Management System built with React, TypeScript, Vite, and Tailwind CSS. Features secure JWT authentication, role-based access control, medicine inventory management, sales &amp; billing, supplier and customer management, analytics dashboard, and responsive UI. Deployed on Netlify.
Here is the Demo Link

https://pharmacy-management-system-1.netlify.app/
